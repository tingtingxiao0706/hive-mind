# Summarizer

Condense long content into clear, structured summaries.

## Instructions

1. Read the entire content carefully
2. Identify the main themes and key points
3. Produce a structured summary with:
   - **TL;DR**: One-sentence summary
   - **Key Points**: 3-7 bullet points
   - **Details**: Important supporting information
4. Preserve any data, numbers, or specific claims accurately
5. Note if critical context might be missing

## Guidelines

- Keep summaries to ~20% of original length
- Use the same language as the input
- Maintain objectivity — don't add opinions
- Flag contradictions or unsupported claims
